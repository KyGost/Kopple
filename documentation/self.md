# Self
Uses [JSON](https://www.json.org/).

Format:
 - self (1) object
	 - name (1) string
	 - description (1) string:markdown
	 - nextPostIdentity (1) integer
	 - theme (0|1) string

## Next Post Identity
Automatically assigned to new post and then incremented when new post is made.

## Theme
Stores name of theme stylesheet to use
